Integritetspolicy – Elpris

Senast uppdaterad: 2026-03-19

Ansvarig

Denna app utvecklas och underhålls av Robert Mellberg. Vid frågor om integritet, kontakta tillsvidare@gmail.com.

Vilken data samlar appen in?

Appen samlar inte in, lagrar eller behandlar någon personlig data. Ingen registrering eller inloggning krävs. Appen använder inga analys-, spårnings- eller annonstjänster.

Hur fungerar appen?

Appen hämtar aktuella elpriser från ett öppet API och visar dem för användaren. Inga uppgifter om användaren skickas till API:et eller till någon tredje part.

Tredjepartstjänster

Appen kommunicerar med ett externt API för att hämta elprisdata. Denna kommunikation innehåller inga personuppgifter. Appen använder inga andra tredjepartstjänster som samlar in data.

Behörigheter

Appen kräver internetåtkomst för att hämta elprisdata. Inga andra behörigheter begärs.

Datalagring

Appen lagrar ingen data lokalt eller i molnet utöver eventuella cachade elpriser för att minska nätverkstrafik. Cachad data innehåller inga personuppgifter.

Barns integritet

Appen riktar sig inte till barn under 16 år och samlar inte medvetet in data från någon användare, oavsett ålder.

Ändringar i denna policy

Om denna integritetspolicy ändras publiceras den uppdaterade versionen på denna sida med ett nytt datum.
